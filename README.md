# 3-praktice
1. Көліктердің әрқайсысы қозғалады. Автокөлік жол бойымен қозғалады, пойыз рельс бойымен, ұшақ ауада ұшады. Сізге қажет: әр көліктің қозғалысын көрсету, максималды жылдамдығын шығару және жүк немесе жолаушы сыйымдылығын көрсету.
2.	Сандар тізбегі есептерінде базалық класс – тізбек, туындысы арифметикалық, геометриялық және Фибоначчи тізбегі. n-ші мүшесін шығару, бірінші k мүшесін көрсету және мүшелер қосындысын есептеу талап етіледі.
